TABLA DE PILOTES (MURO PANTALLA)


Mediante este menú es posible adaptar la distribución de los pilotes a un plano de planta.
Permite también añadir una tabla para los centros de las cuñas de neopreno de las vigas de la cubierta de vigas prefabricadas, con una herramienta para extraer estos datos a partir de un símbolo de la cartografía.



 Utilizando el botón [Extraer] se busca en la cartografía el tipo de símbolo indicado en este menú, creando una tabla de Pks de los pilotes para el lado derecho, izquierdo y la mediana.
Se puede definir junto a cada dato de PK una distancia d para desplazar el pilote hacia la derecha desde su posición teórica.

Los pilotes se pueden añadir indicando su Pk manualmente o de forma interactiva. Si los Pks se asignan de forma interactiva, el Pk seleccionado puede coincidir con el perfil extraido o con uno interpolado.

Si se activa [Utilizar esta tabla] y en un tramo del eje hay definido un muro con pantalla de pilotes, Istram utilizará los pilotes definidos desde este menú.

La segunda parte del menú, TABLA DE ALTURAS DE PILOTES, permite definir por tramos las alturas totales de los pilotes. Este campo prevalece en caso de existir datos en ambas partes del menú.

Si no existe dato para un PK se sigue utilizando la altura del muro.
