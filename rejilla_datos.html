REJILLA DE DATOS

ISTRAM®/ISPOL® dispone de una rejilla de datos que permite visualizar, ordenar y editar datos procedentes de diferentes partes del programa (listados, selección de CRS, repaso de normativas, proyecciones,...):



Ordenación de datos

Para ordenar filas por columna basta con tener una columna seleccionada y pulsar los botones  (orden ascendente) o  (orden descendente), opciones disponibles también desde el menú Datos.

Si estos botones o el menú no están disponibles, entonces se puede pulsar el botón secundario del ratón sobre la cabecera de la columna que se quiere ordenar. Para invertir el orden basta con hacer clic dos veces sobre la cabecera de una columna. Esta forma de actuar también está disponible cuando están presentes los botones de ordenación y el menú.

El proceso de ordenación es el siguiente: Se comparan las filas por el campo mediante el que se quieren ordenar, en caso de igualdad de los datos de las celdas, se recorren todas las columnas de izquierda a derecha comparando el resto de celdas de las dos filas para decidir el orden.


Combinación de celdas

Una pareja de botones permite combinar dos celdas horizontalmente o separar dos celdas que hayan sido combinadas


Guardado de datos

Para guardar los datos en un fichero ASCII (.txt), se debe acudir a la opción Archivo → Guardar como.... Los datos también se pueden guardar en formato .pdf, .doc, .rtf, es decir, con el mismo formato que se generan los listados.
