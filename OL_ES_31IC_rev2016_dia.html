# DISTANCIA DE PARADA:  Dp = (V * Tp/3.6) + (V*V / (254*(Fl + i)) )                #
# En la tabla se ha utilizado i=0   (inclinacion de la rasante en tanto x uno)     #
#----------------------------------------------------------------------------------#
# VALORES ADMISIBLES (deseables): Se han calculdo para un incremento de velocidad  #
# de 20 Km/h. La norma 2015 no considera valores deseables.                        #
#----------------------------------------------------------------------------------#
# ACUERDO PARA ADELANTAMIENTO : Solo Para Acuerdos Convexos y Calzadas Unicas      #
# Se utiliza la distancia de Adelantamiento y se sustituye en la formula           #
# H2 del obstaculo por 1.1                                                         #
####################################################################################
# KA0 : ACUERDOS MINIMOS (A.LARGOS) : Segun las FORMULAS :                         #
#----------------------------------------------------------------------------------#
# ACUERDO CONCAVO: Kv= (Dp*Dp) / ( 2 * ( HF - H2 + Dp*Tg(ALF)) )                   #
# ACUERDO CONVEXO: KV= (Dp*Dp) / ( 2 * (sqrt(H1) + sqrt(H2))^2 )                   #
# Siendo Dp = (V * Tp/3.6) + (V*V / (254*(Fl + i)) )                               #
# Tomando Fl de la tabla sgun la velocidad y haciendo i=0 (segun la Norma)         #
#----------------------------------------------------------------------------------#
  KA0U   1           : 0->No usar estas formulas  1->Usar estas formulas           #
####################################################################################
# KA1 : ACUERDOS MINMOS (A.CORTOS) : Comprobacion de LONGITUD  L >=  KA1 * Vp      #
#----------------------------------------------------------------------------------#
# Si la longitud del acuerdo es menor que KA1 * V, se incrementa el KV hasta que   #
# cumpla ese valor:    KV = L / A = KA1 * Vp/ A   siendo:                          #
# KA1: un coeficiente configurable, segun la norma española 1.00                   #
#   A: Valor absoluto de la diferencia de pendientes, en tanto por uno             #
#  Vp: velocidad de Proyecto                                                       #
# 5.3.2.2 Pag65
#----------------------------------------------------------------------------------#
  KA1U   1           : 0->No usar estas formulas  1->Usar estas formulas           #
  KA1    1.00        :   1.00 segun Norma española                                 #
####################################################################################
# KA2 : ACUERDOS MINIMOS /A.CORTOS) : Comprobacion de LONGITUD > Dp                #
#----------------------------------------------------------------------------------#
# Si la longitud del acuerdo es menor que Dp, se utilizan las formulas para        #
# acuerdos cortos:                                                                 #
# ACUERDO CONCAVO: KV= 2*Dp/A + (2 * ( HF - H2 + Dp*Tg(ALF)) ) / (A*A)             #
# ACUERDO CONVEXO: KV= 2*Dp/A + (2 * (sqrt(H1) + sqrt(H2))^2 ) / (A*A)             #
# Siendo :                                                                         #
#   A: Valor absoluto de la diferencia de pendientes, en tanto por uno             #
#----------------------------------------------------------------------------------#
  KA2U   1           : 0->No usar estas formulas  1->Usar estas formulas           #
####################################################################################
