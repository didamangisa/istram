PLANTILLAS PARA PLANOS DE TRANSVERSALES: LEYENDA

Mediante este menú me permite el programa generar una leyenda mediante elementos lineales o rellenos:


Podremos seleccionar un tipo de línea para la leyenda. Si escogemos el modo relleno se recomienda utilizar un tipo de línea sombreada (L72, L73...)
  • Dist.horiz.: desplazaremos el dato horizontalmente.
  • Dist. vert. : desplazaremos el dato verticalmente.
  • Longitud: podremos dar una mayor o menor longitud al tipo de línea mostrada en la leyenda.
  • Ancho: podremos asignar un espesor a cada dato de la leyenda
  • Estilo: formato del texto
  • Texto: diferentes palabras que podemos escribir para nuestra leyenda
