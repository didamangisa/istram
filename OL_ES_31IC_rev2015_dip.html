# AJUSTE POR NUMERO DE CARRILES QUE GIRAN CON EL PERALTE -------------------------- #
# La formula de calculo de la longitud de transicion de peraltes por control de la  #
# pendiente relativa de borde, lleva un factor de ajuste por el numero de carriles  #
# que giran, de modo que Lp = P*NC*AC*ACfac/IpMax (AC es el ancho de un carril)     #
# El numero de carriles NC es el numero de carriles desde el Eje de Giro al borde   #
# de calzada. Si tenemos el caso de calzada con 3 carriles y eje de giro en el      #
# centro de esa calzada, NC=1.5. Los valores por defecto son NC=2 para autopistas y #
# NC=1 para carreteras. AC=3.5 m. Si se dan en esta tabla al leerla se pasan a la   #
# pestaña GENERAL. Los valores de la pestaña GENERAL son los que se usarán.         #
# El numero de carriles puede darse en una tabla por Pk si hay cambios a lo largo   #
# del eje.                                                                          #
#  Num Carri NC    Fact Ajus ACfac                                                  #
#  ------------    ---------------                                                  #
   TA  1.0               1.00                                                       #
   TA  1.5               0.83                                                       #
   TA  2.0               0.75                                                       #
   TA  2.5               0.70                                                       #
   TA  3.0               0.67                                                       #
   TA  3.5               0.67                                                       #
