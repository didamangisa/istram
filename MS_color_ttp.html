COLOR POR TTP

Permite realizar un mapa de colores a partir de las cotas dadas por un fichero procedente de una triangulación (.ttp). El mapa de colores será generado como líneas que podrán ser guardadas en un árchivo *.edm.



El primer paso será cargar el fichero .ttp procedente de una triangulación desde el menú de control de ttp del cual tenemos una llamada directa mediante la casilla .

Una vez cargado, el programa automáticamente identificará la Z mínima y la Z máxima. Valores que estarán a nuestra disposición para establecer el tramo de cotas que posteriormente dibujaremos.
Mediante la casilla Previo tendremos una visualización dinámica de como podría quedar nuestro dibujo con los diferentes colores que hayamos establecido.

En función de los valores de cota el programa establece un Intervalo de cotas a partir del cual definiremos un tipo de línea sólida. El rango de cotas lo podremos establecer mediante Intervalo de cota o mediante número de tramos. Ambas opciones conmutan entre sí.

Cuando el archivo *.ttp representa las cotas de una superficie, se pude generar un listado de areas 2d y 3d para cada intervalo de cotas y el total, para ello hay que pulsar en el botón Areas (TTP de cotas).

Cuando el archivo *.ttp representa espesores o profundidades (ttp para isopacas) se puede generar un listado de area 2d y volúmen para cada intervalo de espesores y el total, para ello hay que pulsar en el botón Volúmenes (TTP para Isopacas).

Se añade una nueva utilidad para calcular una curva de llenado en el caso de un TTP para isopacas.

Cuando el archivo *.ttp representa espesores o profundidades (ttp para isopacas), pulsando sobre el botón C.Llenado (TTP para Isopacas), .se calcula una curva de llenado. En este caso se suman a parte los intervalos negativos y los positivos. También hay que tener en cuenta que las áreas 2D son acumulativas desde cero hacia los intervalos positivos y desde cero hacia los intervalos negativos. P.e. Si tenemos un TTP con cotas entre 10 y 30.  El area 2D para el volúmen entre 15 y 16  contiene todas las areas 2d de los triángulos entre 15 y 30.



Una vez hayamos definido los tipos de línea sólida para cada intervalo de cota estaremos en disposición de Generar el dibujo del mapa de colores.



Una vez generado podremos incorporar al diujo una leyenda mediante el comando Crear, pudiendo modificar el tipo de texto, el Modelo o el Título que aparecerá en el cuadro de cotas.

A partir del comando Gama de colores podremos indicarle al programa el tipo de línea inicial de nuestro intervalo y el número de líneas que lo componen.



Podremos Cargar o Guardar la información asociada a los intervalos.
