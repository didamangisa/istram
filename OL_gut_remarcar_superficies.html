PLANTILLAS PARA PLANOS DE TRANSVERSALES: REMARCAR SUPERFICIES

Permite recalcar con un tipo de línea una determinada zona de una superficie dada entre unos códigos inicial y final. Podremos indicarle  el lado del eje que queremos remarcar, ambos lados, derecho o izquierdo.

Las superficies definidas para Recintos quedan disponibles para poder ser remarcadas desde este submenú.
