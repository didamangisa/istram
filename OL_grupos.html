Grupos de ejes

Despliega un menú que permite dar nombres a los grupos de ejes que se ha definido en la planta. Así mismo, este menú permite activar/desactivar algunos grupos para inhibir su cálculo y representación en los cálculos del proyecto completo. La configuración en cuanto a la activación/desactivación y los nombres de los grupos de ejes puede guardarse o cargarse mediante ficheros .gru.

Desde esta ventana también se pueden ver los nombres de los diversos ejes que componen cada grupo, siendo posible cambiar el grupo y el nombre de los ejes; para que los cambios sean definitivos aparece la opción [Guardar <proyecto>.cej]. (la asignación del grupo al que pertenece un eje se almacena en el archivo .cej).

Se puede asignar a cada grupo un eje diferente para proyectar los volúmnenes en los archivos pkcvXX.res. Si el valor se deja a 0, se utiliza el eje poara proyectar volúmenes general de la tabla de Proyecto.
 

Se puede definir para cada grupo una tabla .dar, en la columna Tabla .dar.

Se puede definir para cada grupo una tabla .dar para firmes, en la columna .dar firmes.

Se puede definir para cada grupo una sección tipo mediente un archivo .vol, en la columna Sección .vol tipo. Si el eje tiene una propia definida mandaría la del eje, sino se utiliza la del grupo.

Se puede definir para cada grupo un nombre de base para los archivos .vol en la columna Nombre de base y si se define aquí, este manda sobre el nombre de base general.

Esta información de los grupos se guarda/recupera en el archivo .pol del proyecto.

Al lado del nombre de cada eje aparece una casilla que por defecto está de color cyan. Si se pulsa en esta casilla aparece una ventana de selección de color que permite asignar un determinado índice de color a cada eje y por ende a cada grupo, pudiendo de esta manera diferenciarse los diferentes grupos por colores en los menús de PLANTA, ALZADO y COMPLETO, y en el modo No decorado, pues las líneas de tipo 53 (ejes) se dibujan con el color aquí definido.

El botón [Agrupar por .dar] permite agrupar los ejes segun la tabla de cubicación asociada a cada eje. Esto es útil a la hora de generar listados con mediciones conjuntas (cvoltot.res, cvolgru.res, cvolejes.res,...).

En la parte inferior de este menú, zona izquierda, se pueden ver todos los ejes del proyecto. En ella se pueden hacer las siguientes acciones:

  • Columna Eje: Muestra el número de todos los ejes del proyecto. Se puede acceder al eje, tecleando su número en la primera casilla de esta columna.
  • Columna Nombre:  Muestra el nombre de todos los ejes del proyecto. Este nombre puede ser cambiado directamente en esta casilla.
  • Columna Bloqueado:  Muestra si este eje está bloqueado. Un eje se puede marcar como bloqueado (también se pueden bloquear ejes desde la pestaña GENERAL). Un eje bloqueado puede seleccionarse, pero no es posible modificar su geometría.
  • Columna Grupo: Muestra el grupo al que pertenece cada eje del proyecto. Este grupo puede ser cambiado directamente en esta casilla.
En la parte inferior de este menú, zona derecha,  se muestran todos los ejes del grupo actual, siendo el grupo actual grupo visible primer lugar en la parte superior de este menú Grupos. El grupo actual se puede ir cambiando moviendose con la barra de deslizamiento de la parte superior de este menú de Grupos.

El botón [Renombrar Ejes] permite renombrar todos los ejes de un grupo haciendo referencia al PK en el que se proyecta su punto inicial (o final) sobre un eje principal.
