#---------------------------------------------------------------------------------------#
# Descripción de la formulacion aplicada :                                              #
#                                                                                       #
# Los datos suministrados a ISPOL para que pueda ofrecer al usuario los parámetros      #
# de diseño son resultado de aplicar la formulación comunmente aplicada en proyectos    #
# ferroviarios y que aparece documentada en diversas publicaciones técnicas, como es el #
# caso de la norma RENFE, ADIF y AENOR (que transpone la norma ENV 13803-1 europea)     #
#                                                                                       #
# Estos parámetos son agrupados por tramos de aplicación de velocidad, pudiendose       #
# definir modos (en principio normal y excepcional, aunque pueden definirse hasta 10)   #
# Todos estos tramos son agrupados en categorias, que pueden ser elegidas así desde la  #
# ventana de diseño en planta .                                                         #
#                                                                                       #
# Asímismo, puede definir una tabla de datos adicional con parámetros más restrictivos  #
# para controlar los radios mínimos y longitudes de clotoide. Los datos de la tabla     #
# serían interpolados para velocidades intermedias                                      #
#                                                                                       #
# La aplicacción de PERALTES es efectuada de manera automática por ISPOL, sin embargo   #
# puede usar la tecla de peralte obligado e introducir el que considere conveniente.    #
# En este caso, ISPOL ofrece un pequeño resumen para que puedan observarse los límites  #
# de aplicación (mostrandose los valores dinámicos de ANC, Hi, etc )                    #
#                                                                                       #
# Posteriormente desde alzado puede rellenar automáticamente los perales pulsando la    #
# telca 'AUTO DIP' que recogerá los datos automáticos asociados a cada radio y los      #
# obligados que se hayan definido (quedando constancia además de su existencia)         #
# Tambien puede aplicar las tablas de peralte por velocidad que desee.                  #
#                                                                                       #
# K            : K    = s/(g*3.6*3.6)   Constante, depende del ancho                    #
# RADIO MINIMO : Rmin = K             * (Vmax * Vmax )/(hi+hmax)                        #
#                                                                                       #
# PARA CADA VALOR DEL RADIO R:                                                          #
#               H TEORICA    : ht   = K * (Vmax*Vmax)/R                                 #
#                            : htr  = ht - hi;                                          #
#               H REAL       : hr   = Si(hmax < htr)->hmax  Sino->htr                   #
#               H LEY PERALTE: hp   = Si(hmax < htr)->hmax  Sino->hmax*Rmin/R           #
#                    Kapli>0 : hk   = Kapli*K*(Vmax*Vmax)/R                             #
#               H PRACTICO   : hp   = Si(hmax < hk)->hmax  Sino->hk                     #
#                                                                                       #
#               Si el valor de Kapli es distinto de cero el peralte practico o aplicado #
#                se calcula de la forma descrita, de lo contrario se sigue una ley de   #
#                peralte proporcional.                                                  #
#                                                                                       #
